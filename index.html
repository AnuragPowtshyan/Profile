Skills

HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure and content of a website, using tags to define elements like headings, paragraphs, images, and links. HTML forms the foundation of any web page.

CSS

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in a markup language like HTML. It controls the visual appearance of a website, including layout, colors, fonts, and other styling aspects, separating content from presentation.

JavaScript

JavaScript is a versatile scripting language primarily used to add interactivity and dynamic behavior to websites. It enables client-side scripting, allowing for features like animations, form validation, and dynamic content updates, enhancing the user experience.

Python

Python is a versatile programming language widely used for data analysis. Utilizing libraries like pandas for data manipulation, numpy for numerical computations, matplotlib, and seaborn for data visualization, it enables powerful and efficient data handling and insights.

Power BI

Power BI is a business analytics tool by Microsoft that enables users to visualize data, share insights, and make data-driven decisions through interactive reports, dashboards, and data visualizations. It's designed for ease of use and integration.

MySQL

MySQL is a popular open-source relational database management system (RDBMS) used for storing and managing structured data It's reliable, performant, and widely used in web applications.

Excel

Excel is a spreadsheet software by Microsoft that allows users to organize, format, and calculate data with formulas. It includes features like pivot tables, charts, and graphs, making it essential for data analysis and management.
